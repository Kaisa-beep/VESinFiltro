Tor Browser

Tor Browser es un navegador gratuito de código abierto configurado para funcionar en la red Tor, en la que tus páginas pasan encriptado por varios servidores antes de salir a internet.

Es un poco más lento que Hotspot Shield pero te da privacidad

Descarga disponible para Windows Mac Linux

Instalación

  1. Abre el archivo descargado para iniciar a descomprimirlo
  2. Abra la carpeta descomprimida
  3. La aplicación está en esa carpeta lista para usarse, la puedes mover a otra carpeta o pen drive

Conexión con Tor Browser

  1. La aplicación haciendo doble click sobre ella
    En Windows la aplicación se llama Start Tor Browser o Start Tor Browser.exe
  2. Se abrirá una ventana mostrando el estado actual mientras te conecta a la red de Tor de manera anónima
  3. Una vez conectado el programa abrirá automáticamente un Navegador de Internet. El navegador de Tor Browser no guarda historial una vez se cierre por tu seguridad, al cerrar todas sus ventas te desconectará de Tor
Tor Browser Bundle sólo evade bloqueos y protege tus comunicaciones al usar el navegador incluido
Recuerda que cualquier herramienta para evadir bloqueos o aumentar tu privacidad podría ponerte en peligro si es maliciosa, usa herramientas probadas y conocidas, descarga sólo desde la fuente o alguien de confianza.
Algunas de estas herramientas proveen mayor privacidad ante el gobierno y proveedor de servicio, pero no son infalibles y puede ser posible que se detecte su uso
