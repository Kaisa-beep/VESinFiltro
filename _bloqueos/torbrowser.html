Tor Browser

Tor Browser es un navegador gratuito de código abierto configurado para funcionar en la red Tor, en la que tus páginas pasan encriptado por varios servidores antes de salir a internet, ofuscando su origen para mejorar tu privacidad y sirve para evadir los bloqueos en internet.

Es un poco más lento que otras herramientas, pero además de evadir la censura ofusca el origen del trafico

Descarga disponible para Windows Mac Linux Otras opciones

Instalación

  1. Abre el archivo descargado para iniciar a descomprimirlo
  2. Abra la carpeta descomprimida
  3. La aplicación está en esa carpeta lista para usarse, la puedes mover a otra carpeta o pen drive

Conexión con Tor Browser en CANTV

Tor ha sido bloqueado en CANTV, usar las instrucciones para evadir el bloqueo en CANTV, o usa la conexión estándar en otros proveedores

  1. Abre la aplicación haciendo doble click sobre ella
    En Windows la aplicación se llama Start Tor Browser o Start Tor Browser.exe
  2. Se abrirá una ventana mostrando el estado actual mientras te conecta a la red de Tor de manera anónima
  3. Esta conexión fallará en CANTV si no se ha configurado especialmente, seleccione la opción configurar.
    A partir de aquí tines dos opciones:
    Opción A: Utilizar un puente no bloquedo
    1. Visitar página con puentes de Tor no públicos bridge.torbroser.org (recomendado: tipo obfs4)
    2. Copia la lista de puentes
    3. En la configuración de conexión seleccionar “Tor está censurado en mi país”
    4. Pegar los puentes en la sección "Colocar un puente que conozco”
    Opción 2, utilizar un puente tipo meek
    1. En la configuración de conexión seleccionar “Tor está censurado en mi país”
    2. Marca la opción "Selecciona un puente integrado"
    3. Elige el puente “meek-Azure"
  4. Una vez conectado el programa abrirá automáticamente un Navegador de Internet. El navegador de Tor Browser no guarda historial una vez se cierre por tu seguridad, al cerrar todas sus ventas te desconectará de Tor
